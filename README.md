🚀 Anas Khan — Personal Portfolio Website
> Live at: **[https://anaskhan200.github.io/portfolio-website](https://anaskhan200.github.io/portfolio-website)**
A clean, dark-themed personal portfolio website built with pure HTML, CSS and JavaScript — no frameworks, no dependencies.
---
✨ Features
⚡ Typed text animation cycling through roles
🎨 Dark space theme with purple + teal accent system
📱 Fully responsive — works on mobile and desktop
🌀 Scroll-triggered fade-in animations
🔢 Real ML project metrics showcased in hero section
🧩 Sections: Hero, Skills, Projects, Certification, Education, Contact
---
🛠️ Built With
HTML5 — semantic structure
CSS3 — custom properties, grid, flexbox, animations
Vanilla JavaScript — typed effect, IntersectionObserver
Google Fonts — Space Grotesk + JetBrains Mono

📁 Project Structure
```
portfolio-website/
```

🔧 How to Run Locally
Just open `index.html` in any browser — no build step needed.
```bash
git clone https://github.com/AnasKhan200/portfolio-website.git
cd portfolio-website
# Open index.html in your browser
```
